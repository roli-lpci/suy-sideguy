# Code of Conduct

We follow a simple standard: be respectful, constructive, and security-conscious.

## Expected behavior
- Assume good intent
- Focus on evidence and reproducibility
- Report security concerns responsibly (see `SECURITY.md`)

## Unacceptable behavior
- Harassment or abusive language
- Publishing exploit details irresponsibly
- Intentionally harmful contributions

## Enforcement
Maintainers may moderate, remove, or block contributions that violate this policy.
